Was ist das C-Howto?

Das C-HowTo ist ein kostenloses Lehrbuch. Es steht im Internet unter www.c-howto.de bereit. Es bietet einen leichten Einstieg in die Programmierung anhand der Programmiersprache C. Diese Sprache ist der Grundstein vieler anderer Sprachen. So hat man es später leichter C++, Java, PHP, Javascript oder Perl zu lernen - und dies ist nur eine Auswahl.

Es wird keinerlei Programmier-Kenntnis vorausgesetzt. Wenn man einen PC bedienen kann, ist der Anfang schnell gemacht und man kann sofort loslegen.

Gibt es auch eine gedruckte Version des Lehrbuches?

Ja. Über die Interseite kann man sich einen Überblick verschaffen und die Inhalte kostenlos nutzen. Wenn der Inhalt einem zusagt und das Lernen mit einem Buch leichter fällt, kann das Handbuch auch in gedruckter Form bestellt werden. Erwerben Sie das C-HowTo für ein paar Groschen auf der Bestellseite.
