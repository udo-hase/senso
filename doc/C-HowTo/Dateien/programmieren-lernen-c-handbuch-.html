Was ist C?

Programmieren kann jeder, so schwer ist das nicht. Gezeigt werden soll dies anhand der Programmiersprache C. Diese Sprache ist in den Grundstruktur recht einfach, bietet aber fasst undendliche Möglichkeiten. In C lässt sich so gut wie alles programmieren. Selbst Windows und Linux wurden in C geschrieben. Jegliche Arten von Software und Hardware-Geräten lassen sich mit C programmieren. C bietet ein sicheres Fundament welches einem viele Türen öffnet. Das Basis-Wissen ist C, welches man für spezielle Programmier-Richtungen wie Hardware-, Linux-, Windows-, Internet-programmierung, etc vertiefen kann.
